# 🚀 How to Run Your AI Transcription Dashboard
## Complete Beginner's Guide (No Technical Knowledge Required!)

---

## 📋 What You Need Before Starting

1. **A Computer** (Windows, Mac, or Linux)
2. **Internet Connection** (for the first setup only)
3. **A Web Browser** (Chrome, Firefox, Safari, or Edge)

---

## 🎯 STEP-BY-STEP INSTRUCTIONS

### STEP 1: Open Your Terminal/Command Prompt

**On Windows:**
1. Press the `Windows Key` + `R` on your keyboard
2. Type `cmd` and press Enter
3. A black window will open - this is your Command Prompt

**On Mac:**
1. Press `Command` + `Space` on your keyboard
2. Type `terminal` and press Enter
3. A window will open - this is your Terminal

**On Linux:**
1. Press `Ctrl` + `Alt` + `T` on your keyboard
2. A window will open - this is your Terminal

---

### STEP 2: Navigate to Your Project Folder

In the terminal/command prompt window, type this command and press Enter:

**On Windows:**
```
cd "f:\Dhruv's\Kolej shi\IBM-BOB\IBM-BOB-Hackathon-Semicolon"
```

**On Mac/Linux:**
```
cd "/path/to/IBM-BOB-Hackathon-Semicolon"
```

💡 **What this does:** This tells your computer to go to the folder where your dashboard files are located.

---

### STEP 3: Start the Dashboard

Now type this command and press Enter:

```
npm run dev
```

💡 **What this does:** This command starts your dashboard application.

---

### STEP 4: Wait for the Dashboard to Start

You will see some text appearing in the terminal. Wait until you see something like:

```
  VITE v5.2.0  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

✅ **This means your dashboard is ready!**

---

### STEP 5: Open the Dashboard in Your Browser

1. Open your web browser (Chrome, Firefox, Safari, or Edge)
2. In the address bar at the top, type: `http://localhost:5173`
3. Press Enter

🎉 **Your AI Transcription Dashboard will now appear!**

---

## 🖥️ What You'll See on the Dashboard

Your screen will be divided into 3 sections:

### **LEFT SIDE:**
- **Upload Audio** - A box where you can drag and drop audio files
- **Action Items** - A checklist of tasks

### **MIDDLE:**
- **Transcript** - The text version of your audio
- **Search Bar** - To find specific words in the transcript

### **RIGHT SIDE:**
- **AI Summary** - Key points from the transcript
- **Topics** - Main subjects discussed

### **TOP:**
- **Export Button** - To download your transcript and summary

---

## 🎮 How to Use the Dashboard

### To Upload an Audio File:
1. Find an audio file on your computer (MP3, WAV, M4A, or OGG)
2. Drag it to the "Upload Audio" box on the left
3. OR click the box and select a file from your computer

### To Search in Transcript:
1. Look at the middle section
2. Type a word in the search box
3. The word will be highlighted in cyan color

### To Check/Uncheck Action Items:
1. Look at the left side, bottom section
2. Click the checkbox next to any item
3. The item will be marked as complete

### To Export Your Data:
1. Click the "Export" button at the top right
2. Choose what you want to export (Transcript, Summary, Action Items)
3. Choose a format (Text, JSON, or Markdown)
4. Click on your preferred format
5. The file will download to your computer

---

## 🛑 How to Stop the Dashboard

When you're done using the dashboard:

1. Go back to the terminal/command prompt window
2. Press `Ctrl` + `C` on your keyboard (works on Windows, Mac, and Linux)
3. The dashboard will stop running
4. You can close the terminal window

---

## 🔄 How to Start It Again Later

Just repeat Steps 1-5:
1. Open Terminal/Command Prompt
2. Navigate to project folder (`cd` command)
3. Run `npm run dev`
4. Open browser to `http://localhost:5173`

## ❓ Common Questions

**Q: What if I see an error when running `npm run dev`?**
A: Make sure you're in the correct folder. Try Step 2 again.

**Q: What if the browser shows "This site can't be reached"?**
A: Make sure the terminal is still running and showing the "ready" message.

**Q: Can I use this on my phone?**
A: Yes! If your phone is on the same WiFi network, you can access it. Ask for help with this.

**Q: Do I need internet to use this?**
A: After the first setup, you can use it offline!

**Q: Where do my exported files go?**
A: They go to your computer's "Downloads" folder.

---

## 🆘 Need Help?

If something doesn't work:
1. Make sure you followed each step exactly
2. Check that you're in the right folder (Step 2)
3. Try closing the terminal and starting over from Step 1
4. Make sure no other program is using port 5173
